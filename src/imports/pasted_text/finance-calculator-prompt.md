# MASTER UI/UX PROMPT — FinanceCalculator.com

You are an expert Product Designer, UX Researcher, UI Designer, Accessibility Specialist, and SEO-focused SaaS Designer.

Your task is to design a world-class financial calculator platform called **FinanceCalculator.com**.

The website should look like a modern fintech product rather than a traditional blog.

The primary goals are:

* Help users solve financial calculations quickly.
* Build trust and credibility.
* Encourage users to explore multiple calculators.
* Support future monetization through non-intrusive advertising.
* Create an interface that is scalable to hundreds of calculators.
* Optimize engagement and user retention.
* Maintain excellent accessibility and performance.

---

## Design Philosophy

The design must be:

* Minimal
* Elegant

Near buttons

Blocking content

Interrupting user interaction
